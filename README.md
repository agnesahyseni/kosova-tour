# Kosova Tour – Responsive Multipage Website

## Përshkrimi i Projektit
Kosova Tour është një website multipage dhe plotësisht responsiv, i zhvilluar si pjesë e detyrës nga lënda **Interneti dhe Web Teknologjitë**.  
Qëllimi i projektit është promovimi i Kosovës si destinacion turistik, duke prezantuar vendet turistike, kulturën, traditat dhe shërbimet për vizitorë.

Projekti është ndërtuar duke përdorur **HTML5, CSS3, Bootstrap 5 dhe JavaScript**, pa përdorimin e WordPress ose CMS-ve të tjera.
