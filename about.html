About This Project

Kosovo Tour is a multipage responsive website developed as a university project with the aim of presenting Kosovo as a rich destination in terms of culture, history, and tourism. The website is designed to inform visitors about major cities, regions, and natural attractions through a clean layout, structured content, and interactive elements.

This project focuses on combining informative content with modern web technologies to create an engaging user experience. Special attention has been given to responsiveness, accessibility, and usability across different devices and screen sizes.

Project Goals

  • Develop a multipage responsive website with at least 10 pages.
  • Use a personalized structure with folders for CSS, JS, and images.
  • Create an interactive and user-friendly interface using JavaScript functionalities like modals, alerts, and carousels.
  • Implement Bootstrap components such as responsive navbar, carousel, buttons, alerts, cards, and modal windows.
  • Include essential elements like headings, lists, tables, images/videos, and contact forms with HTML5 validation.

Technologies Used

HTML5, CSS3, JavaScript, and Bootstrap for responsive layout and interactive components. A unique font was chosen to enhance design.

Student Information

This project was developed by Agnesa Hyseni, a first-year student in Computer Science and Engineering, as part of academic coursework requirements.
